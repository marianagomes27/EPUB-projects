Abstract

O presente trabalho de casa consistiu, primeiramente, em realizar o XDS (XML Schema Definition) para dois ficheiros XML, fornecidos pelo docente: CLAV-processos-arvore e CLAV-processos-info-completa. De seguida, foi-nos proposto analisar ambos os schemas, comparando-os e tirarando conclusões acerca das mesmos.

Inicialmente, procedeu-se à análise dos XML's fornecidos, que continham as informações necessárias à construção de cada um dos schemas. Os schemas permitiram, então, representar as caraterísticas de cada elemento, bem como a relação entre os elementos e ainda os atributos associados aos mesmos. Os pricipais tipos de elementos/atributos encontrados foram os seguintes: string, boolean, ID, array, object, number, etc.

Fazendo uma análise global relativamente a cada um dos schemas construídos, foi possível concluir sobre as principais diferenças entre ambos. Desta forma, destaca-se como principal diferença, o facto de na constituição do XSD CLAV-processos-arvore estar presente um elemento "filhos", sendo que o mesmo não se encontra no segundo XSD. Este elemento é do tipo array, o que lhe permite armazenar uma coleção de elementos, de modo a que cada elemento possa ser identificado por um index (indice), o que faz com que a sua inserção seja vantajosa no sentido em que permite uma melhor organização estrutural tanto do XML como do XSD.
