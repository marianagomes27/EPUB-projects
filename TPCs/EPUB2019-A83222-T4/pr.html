Abstract

O presente trabalho de casa consistiu em associar a um documento XML (Extensible Markup Language) um documento XSL (Extensible Stylesheet Language), criando, consequentemente, um ficheiro HTML (HyperText Markup Language) para o mesmo.

Inicialmente, analisou-se o ficheiro XML fornecido pelo docente, que contém um arquivo de sonoro de músicas portuguesas. De seguida, foi criado um XSLT (Extensible Stylesheet Language for Transformation) ao qual se associou o XML em questão, no qual foram definidos os parâmetros de formatação pretendidos para a página HTML e onde foram também organizados os dados de cada música.

Por fim, foi então criada a página web pretendida, com a estrutura e organização desejadas. Todos os ficheiros mencionados anteriormente encontram-se na secção "Deliverables" .
